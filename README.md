# 🌴 Kerala Tourism Assistant

An AI-powered Kerala Tourism Chatbot built using **Streamlit**, **FAISS**, **Sentence Transformers**, and **Mistral AI**.

## Features

* 🤖 AI-powered tourism assistant
* 🗺️ Kerala tourism information
* 🔍 FAISS vector search
* 👤 User registration and login
* 💬 Interactive chat interface
* 🎤 Voice input support

## Technologies Used

* Python
* Streamlit
* FAISS
* Sentence Transformers
* Mistral AI
* Pandas

## Installation

Clone the repository:

```bash
git clone https://github.com/smerats/KERALA-TOURISM-ASSISTANT.git
```

Go to the project folder:

```bash
cd KERALA-TOURISM-ASSISTANT
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
